# GBEMU

This is a gameboy emulator written in c :)

Currently not implemented:
 - LCD stat interrupt
 - Serial
 - Window
 - Sound
 - Proper timing
 - MBC

Other features to implement:
 - Load rom from file
 - Save states

# Build / Run

Ensure make & openGL are installed.

1. ```make build```
2. ```make run args=<rom_filename>```
